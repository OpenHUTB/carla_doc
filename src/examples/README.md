## 结束进程

客户端连接不上服务端的默认2000端口
```shell
netstat -ano | findstr "2000"
taskkill /PID 35320 /F
```


服务端指定端口启动:
```shell
CarlaUE4.exe -carla-rpc-port=3000
```
指定端口连接服务端：
```shell
-p 3000
```
