  1. 按Ctrl+Shift+P，输入Python:Select Interpreter并选择；
  2. 选择或创建所需的Python解释器；
  3. 单击行号左侧设置断点；
  4. 点击右上角三角箭头，选择Python Debugger: Debug Python File；
