解决上传出熬过100M大文件的问题

参考

git上传超过100MB的大文件

  1. 安装LFS
git lfs install
  1. 上传文件
git lfs track "neuro.pdf"
  1. 添加到 .gitattributes 区
git add .gitattributes

  1. 配置 lfs 仓库地址

    git config lfs.https://gitee.com/dong_666/neuro/info/lfs.locksverify false

  2. 添加到暂存区

    git add .

  3. 将暂存区内容添加到仓库中 

    git commit -m "add big file"

  4. 删除./git/hooks/pre-push文件

    rm .git/hooks/pre-push

  5. 上传远程代码并合并

    git push origin master
# 真正推送大文件
git lfs push --all origin

注意

需要是企业版或企业版的试用版，否则还是会出现不能上传超过100M大文件的问题。
